# Agent notes

Read each of these before working on the project. They are not optional
background — each one carries information you cannot derive from the code
alone.

- [maket_programy.md](maket_programy.md) — technical blueprint: stack
  versions, DB schema, cross-platform notes. The source of truth for "what
  this project is supposed to be".
- [TODO.md](TODO.md) — task tracker keyed against `maket_programy.md`, with
  status legend (`[x]`, `[~]`, `[!]`, etc.). Check this before starting any
  feature to see whether it's already done, in flight, or deliberately
  out of scope.
- [BugHunt.md](BugHunt.md) — how to use the headless debug-UI harness in
  `MusicApp.BugHunt/`: API surface, where artifacts land, and the bug
  taxonomy to report against. Use this whenever you need to actually run
  the app, take a screenshot, or inspect the visual tree.
